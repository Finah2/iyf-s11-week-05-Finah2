Week 5: DOM Manipulation

This week I learned to make pages actually respond to what you do. Click things, type things, watch things change.

📚 Open Task Manager 🔬 DOM Practice

What I Built This Week

🎯

Selecting Elements

Learned four ways to grab elements — getElementById, querySelector, querySelectorAll, and getElementsByClassName. Each one has its place.

🌳

Traversing the DOM

Navigated up and down the DOM tree using parentElement, children, nextElementSibling. The DOM is literally a tree and you can climb it.

✏️

Modifying Content

Changed text with textContent, swapped HTML with innerHTML, updated attributes and styles — all without touching the HTML file.

➕

Adding & Removing Elements

Created elements with createElement, added them with appendChild and prepend, cloned existing ones, and removed them with .remove().

👂

Event Listeners

Made things respond to clicks, key presses, form submissions, mouse movements. Built a click counter and keyboard shortcuts including Ctrl+S and Escape.

🫧

Event Bubbling & Delegation

Learned how clicks travel up the DOM from child to parent. Used event delegation to handle all task clicks with just ONE listener on the parent — much cleaner.

✅

Form Validation

Added real-time validation to forms — name length check, email format check with regex, error messages that appear and disappear as you type.

📚

School Task Manager

Built a full to-do list with add, complete, delete, filter and edit features. Pre-loaded with real school tasks. My most complete project yet.

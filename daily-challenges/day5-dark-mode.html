Dark Mode Toggle

Day 5 — Week 5

Toggles dark mode by adding and removing a class on the body element. CSS handles all the visual transitions — JavaScript just flips the switch.

☀️ Light 🌙 Dark Light Mode

What I learned this week

The DOM is a tree of objects that JavaScript can reach into and change. Every element, attribute and piece of text is a node — and you can create, modify or remove any of them.

Most interesting concept

Event delegation — instead of adding a listener to every item in a list, you add one listener to the parent and let events bubble up. New items automatically work without extra code.

Biggest challenge

Understanding how this dark mode toggle works. The JavaScript only adds or removes a class — all the actual styling is handled in CSS using the .dark selector. Clean separation of concerns.
